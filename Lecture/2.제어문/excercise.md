### 1. 5명 시험봤는데 합격자와 불합격자 가르는 결과 만들기 
---
marks = [90, 25, 67, 45, 80]

```python
number = 0 
for mark in marks: 
    number = number +1 
    if mark >= 60: 
        print("%d번 학생은 합격입니다." % number)
    else: 
        print("%d번 학생은 불합격입니다." % number)
```
